## 🔧 Featured Project — Intelligent Tutoring System

A full-stack AI algebra tutor built on **React · FastAPI · MongoDB · SymPy · OpenAI**.

- **Deterministic-first grading** — exact match → conceptual normalization → SymPy equivalence before any LLM fallback, so symbolic math is graded by code and the model only handles genuine ambiguity.
- **Real-time reasoning validation** — per-step feedback with deterministic guardrails that reject prose-only steps before an OpenAI function-calling analysis compares student work against a reference solution.
- **Resumable assessments** — MongoDB drafts with unique `(user, subtopic)` upserts, nested React state serialization, and knowledge-profile recomputation on finalize.
- **Interactive visual math** — number-line and graph tools that turn canvas actions (points, rays, segments) into structured algebra payloads for the grader.

**Other projects:** a Terraform-provisioned Cloud-Native Web App on AWS (serverless verification via SNS/Lambda/DynamoDB, 46 Jest tests in CI) and an F1 Management System (17-table SQL Server schema, 938-line stored-procedure layer, Django REST + React).
